## 1.4、将blog.sql导入mysql数据库

> windows数据库安装教程：https://blog.csdn.net/weixin_44537669/article/details/114076961



- 1、cmd中登录mysql

~~~mysql
mysql -uroot -p123456
~~~



- 2、查看数据库

~~~mysql
show databases;
~~~



- 3、创建blog数据库

~~~mysql
create database blog;
~~~



- 4、导入blog.sql到blog数据库中 

~~~mysql
mysql -uroot -p123456 blog<blog.sql
~~~



- 5、查看一下：表是否已导入

~~~mysql
mysql -uroot -p123456;   # 登录
use blog;				 # 使用blog数据库
show tables;			 # 显示所有表格
~~~
